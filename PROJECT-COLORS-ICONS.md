# Project Colors & Icons Reference

Use this guide when adding projects via the admin panel.  
All colors and icons below are **active and ready to use** in the admin panel.

| Value (type in admin) | Preview | Suggested Use |
| --------------------- | ------- | ------------- |
| `sky` | 🟦 Sky blue | Cloud, APIs, deployment |
| `violet` | 🟪 Violet | ML/AI, deep learning |
| `emerald` | 🟩 Emerald green | CV, image processing |
| `amber` | 🟨 Amber | Data analysis, NLP |

| `red` | 🔴 Red | Security, anomaly detection |
